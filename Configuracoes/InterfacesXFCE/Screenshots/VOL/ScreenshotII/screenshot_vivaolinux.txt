Arquivo '.gtkrc-2.0'
#####
Utilizado para alterar o tamanho do botão, segue abaixo o código do meu

#####
style "xfce-tasklist-style"
{

# The maximum length of a button before the label ellipsizes.
# When this value is set to -1 the button will expand to the
# entire available space.
XfceTasklist::max-button-length = 120 # pixels. or whatever size you like

}
class "XfceTasklist" style "xfce-tasklist-style" 
#####

Basta criar o arquivo '.gtkrc-2.0' e colocar na pasta home do usuário e reiniciar a máquina.
