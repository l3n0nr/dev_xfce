### INFORMAÇÕES
## GERAIS
Atenção! Esse script contem configurações pessoais do autor, dessa forma é aconselhável revisar o código para visualizar as tarefas realizadas pelo mesmo!

O script esta sendo desenvolvido no Debian Testing(Buster), através dos repositórios padrões do sistema. Assim, poderão haver inconsistências caso seja utilizada outra distribuição.

## ESPECIFICAS
- Função Atualiza: Atualiza(literalmente) o sistema.

- Função Corrige: Corrige programas recorrentes no sistema.

- Função Configura: Realiza configurações na máquina de forma automática.

- Função Limpa: Limpa o sistema, removendo arquivos não necessários.

- Função Instala: Instala programas do gosto pessoal do autor.

- Função Instala outros: Instala programas que precisam de interação do usuário para o prosseguimento do processo.

- Função Remove: Remove programas que não são necessários no sistema.

* - É aconselhável que seja realizado uma verificação básica dentro de caso função, para que não seja realizadas ações inesperadas no sistema do usuário.
